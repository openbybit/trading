# gcore

- backoff: exponential超时计算
- cast: 负责各种类型转换
- conhash: 负责一致性hash计算
- container: HashMap, HashSet
- deadlock: 死锁检测
- encoding: base64, md5 encoding
- env: 效能环境变量封装
- errors: 常见Error封装
- filesystem: 常见文件操作封装
- http: http client封装
- nets: 网络处理函数封装
- sign: 验证功能封装
- timing: 定时器功能
- wildcard: *?字符串匹配
